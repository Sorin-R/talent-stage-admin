Place your admin sidebar icons in this folder.

Recommended file names:
- dashboard.png
- videos.png
- users.png
- comments.png
- reports-queue.png
- reports-archive.png
- analytics.png
- audit-log.png
- admin.png
- moderators.png
- system.png
- settings.png

The app also tries fallback variants automatically:
- snake_case
- no-spaces lowercase
- lowercase with spaces
- exact label with spaces
